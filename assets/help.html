MU310 工具中心 使用說明 (MU310 Tools Center - User Guide)

總覽 Overview

  • ADB 工具 檢查 ADB 環境、列出裝置/COM 埠。
  • 連線修復 一鍵執行 auto_fix_adb_ENG.bat 修復連線。
  • 韌體升級 選擇 .bin 檔，呼叫 Burn_in _611GT.bat 升級。
  • 說明 本頁文件。
  • ADB Tools: check ADB and list devices/COM ports.
  • Connection Fix: run auto_fix_adb_ENG.bat.
  • Firmware Upgrade: select a .bin and run Burn_in _611GT.bat.
  • Help: this document.

安裝與執行 Install & Run

  1. 安裝驅動：若 ADB 無法辨識裝置，請先安裝對應 USB/ADB 驅動。
  2. 將韌體檔放入 FW_IMAGE\ 資料夾（或稍後從瀏覽視窗選取完整路徑）。
  3. 執行 MU310_ADB_TOOLl.exe（或 python main.py）。
  4. 右上角可切換語言與調整字體大小（會影響按鈕、標籤與彈窗）。

各分頁操作 Tabs

ADB 工具 ADB Tools

  • 執行「檢查 ADB 環境」確認安裝與 PATH。
  • 「列出 COM 埠」可列示系統可見的序列埠。

連線修復 Connection Fix

  • 按「執行自動修復」即呼叫 auto_fix_adb_ENG.bat；完成後可回到 ADB 工具檢查。

韌體升級 Firmware Upgrade

  1. 按「瀏覽」選擇 .bin 檔（建議放在 FW_IMAGE\；路徑過長會以中間省略顯示）。
  2. 按「執行韌體升級」；日誌會連續顯示 push/sync/停服務/觸發更新等進度。
  3. 也可直接執行批次檔：
    "Burn_in _611GT.bat" <firmware.bin or full path>

設定 Settings

  • 可自訂 GUI 標題（Title (EN)、中文標題）。
  • 按「儲存設定」即刻生效，並寫入 config.json。
  • 語言切換時會使用對應語系的自訂標題；若空白則退回預設。
注意 Notes
  • 升級期間請勿斷線；裝置可能自動重啟，需數分鐘。
  • 若批次檔顯示找不到檔案，確認檔名與 FW_IMAGE\ 位置或使用完整路徑。

常見問題 FAQ

Q1: 看到 "Firmware file not found"？

請確認檔案在 FW_IMAGE\ 且檔名正確，或於 GUI 重新選擇。

Q2: ADB 無法偵測到裝置？

檢查 USB 線與驅動、裝置的 USB 偵錯模式；於「ADB 工具」分頁按「檢查 ADB 環境」。

Q3: 想重看過往輸出？

所有輸出會自動寫入 logs/。GUI 內的日誌也會連續累加，不會自動清空。

English Summary

Use ADB Tools to verify environment, Connection Fix to auto-run fixes, and Firmware Upgrade to flash via Burn_in _611GT.bat. Put firmware into FW_IMAGE\, select it, then run upgrade. See logs for progress.
