安裝與執行 Install & Run

  1. 安裝驅動：若 ADB 無法辨識裝置，請先安裝對應 USB/ADB 驅動。
  2. 將韌體檔放入 FW_IMAGE\ 資料夾（或稍後從瀏覽視窗選取完整路徑）。
  3. 執行 MU310_ADB_TOOLl.exe（或 python main.py）。
  4. 右上角可切換語言與調整字體大小（會影響按鈕、標籤與彈窗）。
  1. Install drivers: If ADB cannot recognize devices, install corresponding USB/ADB drivers first.
  2. Place firmware files in FW_IMAGE\ folder (or select full path later from browse window).
  3. Run MU310_ADB_TOOLl.exe (or python main.py).
  4. Top-right corner can switch language and adjust font size (affects buttons, labels and popups).

各分頁操作 Tabs

ADB 工具 ADB Tools

  • 執行「檢查 ADB 環境」確認安裝與 PATH。
  • 「列出 COM 埠」可列示系統可見的序列埠。
  • Run "Check ADB Environment" to verify installation and PATH.
  • "List COM Ports" can display system-visible serial ports.

連線修復 Connection Fix

  • 按「執行自動修復」即呼叫 auto_fix_adb_ENG.bat；完成後可回到 ADB 工具檢查。
  • Click "Execute Auto Fix" to call auto_fix_adb_ENG.bat; after completion, return to ADB Tools for verification.

韌體升級 Firmware Upgrade

  1. 按「瀏覽」選擇 .bin 檔（建議放在 FW_IMAGE\；路徑過長會以中間省略顯示）。
  2. 按「執行韌體升級」；日誌會連續顯示 push/sync/停服務/觸發更新等進度。
  3. 也可直接執行批次檔：
    "Burn_in _611GT.bat" <firmware.bin or full path>
  1. Click "Browse" to select .bin file (recommended to place in FW_IMAGE\; long paths will be displayed with middle ellipsis).
  2. Click "Execute Firmware Upgrade"; logs will continuously show push/sync/stop service/trigger update progress.
  3. Can also run batch file directly:
    "Burn_in _611GT.bat" <firmware.bin or full path>

設定 Settings

  • 可自訂 GUI 標題（Title (EN)、中文標題）。
  • 按「儲存設定」即刻生效，並寫入 config.json。
  • 語言切換時會使用對應語系的自訂標題；若空白則退回預設。
  • 可編輯關鍵字顏色設定，讓日誌中的特定文字顯示不同顏色。
  • Can customize GUI title (Title (EN), Chinese Title).
  • Click "Save Settings" to take effect immediately and write to config.json.
  • Language switching will use corresponding language custom title; if blank, fall back to default.
  • Can edit keyword color settings to display specific text in different colors in logs.
注意 Notes
  • 升級期間請勿斷線；裝置可能自動重啟，需數分鐘。
  • 若批次檔顯示找不到檔案，確認檔名與 FW_IMAGE\ 位置或使用完整路徑。
  • Do not disconnect during upgrade; device may restart automatically, taking several minutes.
  • If batch file shows "file not found", verify filename and FW_IMAGE\ location or use full path.

常見問題 FAQ

Q1: 看到 "Firmware file not found"？

請確認檔案在 FW_IMAGE\ 且檔名正確，或於 GUI 重新選擇。

Q1: Seeing "Firmware file not found"?

Please confirm the file is in FW_IMAGE\ with correct filename, or reselect in GUI.

Q2: ADB 無法偵測到裝置？

檢查 USB 線與驅動、裝置的 USB 偵錯模式；於「ADB 工具」分頁按「檢查 ADB 環境」。

Q2: ADB cannot detect device?

Check USB cable and drivers, device USB debugging mode; in "ADB Tools" tab click "Check ADB Environment".

Q3: 想重看過往輸出？

所有輸出會自動寫入 logs/。GUI 內的日誌也會連續累加，不會自動清空。

Q3: Want to review previous output?

All output is automatically written to logs/. Logs in GUI also accumulate continuously, won't clear automatically.

Q4: 關鍵字顏色設定不生效？

編輯關鍵字後點擊「儲存」，顏色設定會立即生效。如果仍有問題，請重新啟動程式。

Q4: Keyword color settings not working?

After editing keywords, click "Save" and color settings will take effect immediately. If still having issues, please restart the program.
